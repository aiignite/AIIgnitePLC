# USR-K 超级网口 PCBA 配置规范

本文档描述焊接在 RH850 PCBA 上的 [USR-K2/K3 超级网口](https://www.usr.cn/Product/21.html) 模块推荐参数，用于 AIIgnitePLC 远程 TCP 部署。

## 硬件连接

- **TTL 串口**：TX/RX 接 RH850 UART3（3.3V 电平）
- **以太网**：RJ45 接入工厂/实验室局域网
- **供电**：3.3V（与 RH850 同板供电）

## 推荐网络参数

| 参数            | 推荐值               | 说明                         |
| --------------- | -------------------- | ---------------------------- |
| 工作模式        | TCP Server           | 后端作为 TCP Client 主动连接 |
| 本地端口        | 8234                 | 避免与 Telnet 默认 23 冲突   |
| IP 地址         | 静态 IP 或 DHCP 保留 | 写入项目硬件配置 `moduleIp`  |
| 最大 TCP 连接数 | 1                    | 避免多客户端争抢             |

## 串口参数

| 参数   | 值     |
| ------ | ------ |
| 波特率 | 115200 |
| 数据位 | 8      |
| 校验位 | None   |
| 停止位 | 1      |

必须与 RH850 UART3 及 DeployPanel 默认一致。

## 透传与打包

| 参数     | 推荐值     | 原因                                                       |
| -------- | ---------- | ---------------------------------------------------------- |
| 注册包   | **关闭**   | ASCII 前缀会破坏 RH850 二进制 CRC                          |
| 心跳包   | **关闭**   | 避免污染 0x55AA55AA 协议流                                 |
| 打包时间 | 0–4 ms     | 减少串口侧延迟                                             |
| 打包长度 | ≥ 600 字节 | 单帧最大约 524B（512B chunk + 头），须小于 K2 串口 RX 800B |

## K2 vs K3 差异

| 项目          | USR-K2               | USR-K3 |
| ------------- | -------------------- | ------ |
| 串口 RX 缓存  | 800 B                | 2 KB   |
| 最大 TCP 连接 | 16                   | 8      |
| 下载 chunk    | 512 B（默认）满足 K2 | 同左   |

## 配置方式

1. **网页**：浏览器访问模块 IP（默认可能为 192.168.0.7），用户名/密码 `admin`
2. **M0 搜索工具**：有人官方配置软件搜索局域网设备
3. **AT 指令**（经调试串口）：
   ```
   AT+SOCK=TCPS,,8234
   AT+UART=115200,8,1,NONE
   AT+ENTM
   ```

## AIIgnitePLC 使用流程

1. 在「设备配置」中为 RH850 CPU 填写 USR-K 模块 IP 与 TCP 端口
2. 在「在线诊断 / 部署」面板选择 **远程 TCP（LAN）**
3. 点击「连接」→「测试连接」（0x6A 状态查询）→「编译并下载」

详细操作与故障排查见 [remote-tcp-deploy.md](./remote-tcp-deploy.md)。

后端须与 PCBA 处于**同一局域网**，通过 `/api/v1/ws/device` WebSocket 桥接至模块 TCP 端口。

## 安全提示

- 不要将 USR 模块 TCP 端口暴露到公网
- 远程工程师应通过 VPN 接入目标 LAN
- 生产环境启用后端 `DEVICE_TCP_ALLOWLIST` 限制可连接 IP 段
